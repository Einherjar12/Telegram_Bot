Домашнее задание — Telegram-боты

Задание 1. Бот-счетчик

Задача: Бот, который считает количество нажатий на кнопку.
Функционал:
•	Кнопка «+1» для увеличения счетчика
•	Показывает текущее значение
•	Кнопка «Сбросить»
База данных:
CREATE TABLE counters (
    user_id INTEGER PRIMARY KEY,
    count INTEGER DEFAULT 0
);

Задание 2. Бот-рандом

Задача: Бот, который выдает случайные числа или делает выбор.
Функционал:
•	Случайное число от 1 до 100
•	Выбор «Орел или решка»
•	Случайный выбор из 2 вариантов
База данных:
CREATE TABLE random_stats (
    user_id INTEGER PRIMARY KEY,
    coins_flipped INTEGER DEFAULT 0,
    numbers_generated INTEGER DEFAULT 0
);

Задание 3. Бот-комплимент

Задача: Бот, который говорит приятные слова.
Функционал:
•	Кнопка «Скажи комплимент» 
•	Сохраняет любимые комплименты
•	Показывает историю комплиментов
База данных:
CREATE TABLE compliments (
    user_id INTEGER,
    compliment TEXT,
    date DATETIME
);

Задание 4. Бот-погода

Задача: Бот, который «предсказывает» погоду.
Функционал:
•	Выбор города из списка
•	«Прогноз» погоды (случайный)
•	Сохранение любимого города.
База данных:
CREATE TABLE weather_prefs (
    user_id INTEGER PRIMARY KEY,
    favourite_city TEXT
);
•	Кнопки: «Москва», «СПБ», «Сочи», «Избранное».

Задание 5. Бот-викторина «Да/Нет»

Задача: Простая викторина с вопросами, на которые можно ответить только «Да» или «Нет».
Функционал:
•	Вопросы с двумя вариантами
•	Подсчёт правильных ответов 
•	Следующий вопрос
База данных:
CREATE TABLE quiz_results (
    user_id INTEGER PRIMARY KEY,
    correct_answers INTEGER DEFAULT 0,
    total_questions INTEGER DEFAULT 0
);

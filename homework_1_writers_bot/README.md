# Домашнее задание №1 — Telegram-бот о писателях 

Задание 1
Создайте базовый каркас чат-бота для Telegram.

Задание 2
Зарегистрируйте своего чат-бота в Telegram.
Установите следующие параметры бота:
■ Название бота;
■ Описание бота;
■ Раздел about;
■ Иконку бота.

Задание 3
Добавьте к боту из второго задания реакцию на данные от пользователя:
■ Если пользователь прислал сообщение «Писатель»,
бот должен ответить «Хемингуэй»;
■ Если пользователь прислал сообщение «Поэт», бот
должен ответить «Шекспир»;
■ Если пользователь прислал сообщение «Книга», бот
должен ответить «Три товарища»;
■ Если пользователь прислал сообщение «Монолог»,
бот должен ответить «Быть или не быть».

Задание 4
Добавьте к боту из третьего задания случайную реакцию на данные от пользователя:
■ Если пользователь прислал сообщение «Писатель»,
бот должен вернуть фамилию писателя. Выбор производится произвольно.
■ Если пользователь прислал сообщение «Поэт», бот
должен вернуть фамилию поэта. Выбор производится
произвольно.
■ Если пользователь прислал сообщение «Книга», бот
должен вернуть название книги. Выбор производится
произвольно.
■ Если пользователь прислал сообщение «Монолог», бот
должен вернуть название известного, литературного
монолога. Выбор производится произвольно.
